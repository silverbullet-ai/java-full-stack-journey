# Module 3 - Collections (ArrayList)

## What is Collections Framework?

A set of classes and interfaces used to store and manipulate data dynamically.

---

## Why Not Arrays?

- Fixed size  
- No flexibility  

---

## What is ArrayList?

- Dynamic array (resizable)
- Part of java.util package
- Allows duplicate elements
- Maintains insertion order

---

## Syntax

```java
import java.util.ArrayList;

ArrayList<Integer> list = new ArrayList<>();
```

---

## Common Methods

- add(value) → add element  
- get(index) → access element  
- set(index, value) → update  
- remove(index) → delete  
- size() → number of elements  

---

## Important Concepts

### Generics

```java
ArrayList<Integer> → stores only integers
```

---

## Key Takeaways

- ArrayList is flexible version of array  
- Automatically resizes  
- Widely used in real applications
